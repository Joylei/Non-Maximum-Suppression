# Non-Maximum-Suppression
**非极大值抑制（Non-Maximum-Suppression，NMS）**是抑制非极大值的元素，可以理解为局部最大搜索。NMS算法在计算机视觉中有广泛应用，特别是目标检测领域。本文将以目标检测举例，讨论NMS算法的实现过程。
